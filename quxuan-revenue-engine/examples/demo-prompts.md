# Quxuan Revenue Engine - Demo Prompts 趣选收益引擎 - 演示提示

## 1. Dynamic Pricing 动态定价 💰

**English:**
> "I run a SaaS product with a base subscription price of $29/month. Can you generate a dynamic pricing strategy considering peak hours (9-17 weekdays), weekend patterns, and upcoming holiday seasons?"

**中文:**
> "我运营一个SaaS产品，基础订阅价29美元/月。请生成动态定价策略，考虑工作高峰时段(9-17点)、周末模式和即将到来的节假日。"

---

## 2. Conversion Funnel 转化漏斗 📊

**English:**
> "Here's my user funnel data: Landing page: 10,000 → Browse: 6,500 → Add to cart: 2,100 → Checkout: 1,200 → Payment: 980 → Completed: 920. Where are users dropping off and how can I improve conversion?"

**中文:**
> "这是我的用户漏斗数据：落地页10000人 → 浏览6500 → 加入购物车2100 → 结算1200 → 支付980 → 完成920。用户在哪里流失？如何提升转化率？"

---

## 3. A/B Test Design A/B测试设计 🧪

**English:**
> "My current checkout conversion rate is 8%. I want to test a new one-page checkout design. Calculate the required sample size to detect a 15% improvement with 95% confidence."

**中文:**
> "我当前的结算转化率是8%。我想测试新的一页式结算设计。计算需要多少样本量才能以95%置信度检测到15%的提升。"

---

## 4. Retention Scoring 留存评分 📈

**English:**
> "Analyze these users for churn risk: User A (last active 20 days ago, 0 logins in 7 days, 0 purchases), User B (last active 1 day ago, 5 logins in 7 days, 3 purchases), User C (last active 5 days ago, 2 logins, 1 support ticket). Predict their LTV if average monthly spend is $50."

**中文:**
> "分析以下用户的流失风险：用户A（20天未活跃，7天0次登录，0次购买），用户B（昨天活跃，7天5次登录，3次购买），用户C（5天未活跃，2次登录，1次客服工单）。假设月均消费50美元，预测他们的LTV。"

---

## 5. Revenue Dashboard 收益仪表盘 📋

**English:**
> "Generate a monthly revenue dashboard with these metrics: Total revenue $45,000, Active users 2,300, New signups 450, Churn rate 4.2%, ARPU $19.56, CAC $35, Average LTV $234. Include trend analysis and recommendations."

**中文:**
> "生成月度收益仪表盘，指标如下：总收入45000美元，活跃用户2300，新注册450，流失率4.2%，ARPU 19.56美元，获客成本35美元，平均LTV 234美元。包含趋势分析和建议。"

---

## 6. Combined Analysis 综合分析 🚀

**English:**
> "I'm launching a premium tier at $99/month for my existing $29/month product. Help me design: 1) pricing strategy for the launch, 2) upsell funnel targeting existing users, 3) A/B test plan for the upgrade prompt timing, 4) retention strategy to prevent downgrade."

**中文:**
> "我要在现有29美元/月产品基础上推出99美元/月的高级版。帮我设计：1）上线定价策略，2）针对现有用户的升级漏斗，3）升级提示时机的A/B测试方案，4）防止降级的留存策略。"
