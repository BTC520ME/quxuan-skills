# 🖥️ 电脑操作指南（小白版）

> 前提：手机上已完成注册（PayPal、Payoneer、Gumroad）
> 总耗时：约30分钟
> 每一步都很短，照做就行

---

## 第1步：下载代码包到电脑

微信把 **趣选v2.1.0_最终优化版.zip** 发给「文件传输助手」，电脑微信打开，保存到**桌面**。

---

## 第2步：解压

桌面上找到 zip 文件 → **右键** → **解压到当前文件夹**

> 没有这个选项？选「全部解压缩」→ 下一步 → 完成

---

## 第3步：装 Git

1. 浏览器打开：**git-scm.com**
2. 点「Download for Windows」
3. 下载完，**双击安装**
4. 一路点 **Next**，什么都不用改
5. 最后点 **Install** → **Finish**

---

## 第4步：配置 Git

1. 按 **Win + R**（键盘左下角）
2. 输入 `cmd` → 回车
3. 黑色窗口里**复制粘贴**下面两行（每行粘完按回车）：

```
git config --global user.name "BTC520ME"
```

```
git config --global user.email "你的GitHub邮箱"
```

> 把「你的GitHub邮箱」换成你真实的 GitHub 邮箱

---

## 第5步：上传代码到 GitHub

在同一个黑色窗口里，**一行一行复制粘贴**，每行粘完按回车，等它跑完再粘下一行：

```
cd %USERPROFILE%\Desktop
```
↑ 等完成 ↓
```
dir
```
↑ 看一眼解压出来的文件夹叫什么名字 ↓
```
cd 解压后的文件夹名
```
↑ 比如 `cd 趣选v2.1.0_最终优化版` ↓
```
git init
```
↑ 等完成 ↓
```
git add .
```
↑ 等完成 ↓
```
git commit -m "趣选v2.1.0最终版"
```
↑ 等完成 ↓
```
git branch -M main
```
↑ 等完成 ↓
```
git remote add origin https://github.com/BTC520ME/quxuan-skills.git
```
↑ 等完成 ↓
```
git push -u origin main --force
```
↑ 弹出登录窗口 → 登录你的 GitHub 账号 → 授权

看到 **done** 就成功了 ✅

---

## 第6步：装 Node.js

1. 浏览器打开：**nodejs.org**
2. 点绿色 **LTS** 按钮下载
3. 双击安装 → 一路 **Next** → **Install** → **Finish**
4. **关掉**黑色窗口，重新按 Win+R → cmd → 回车打开新的

---

## 第7步：装 ClawHub 工具

在新黑色窗口里输入：

```
npm install -g clawhub
```

等它装完（约30秒）。

---

## 第8步：登录 ClawHub

```
clawhub login
```

浏览器会自动弹出来 → 用 GitHub 登录 → 点 **Authorize**

回到黑色窗口看到 **Login successful** 就行了 ✅

---

## 第9步：发布！

**一行一行来**，每行等它显示 `Published successfully` 再输下一行：

```
clawhub publish quxuan-travel
```
```
clawhub publish quxuan-daily
```
```
clawhub publish quxuan-workplace
```
```
clawhub publish quxuan-copywriter
```
```
clawhub publish quxuan-professional
```
```
clawhub publish quxuan-revenue-engine
```

6个都发布成功 → 🎉 **全部搞定！**
