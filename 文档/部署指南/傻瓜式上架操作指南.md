## 第一步：注册 ClawHub 账号 🖥️

1. 打开浏览器，访问 **https://clawhub.com**
2. 点击 **Sign Up**（注册）
3. 用你的 GitHub 账号登录（最简单，点 GitHub 图标一键注册）
4. 完成注册，记住你的用户名

> 💡 ClawHub 发布完全免费，和 npm 一样是开放平台

---

## 第二步：安装 Git 和配置（电脑只需做一次）🖥️

### Windows 电脑：
1. 打开浏览器，访问 **https://git-scm.com/download/win**
2. 自动开始下载，下载完双击安装
3. **安装过程一路点 Next**，所有默认选项都不用改
4. 安装完成后关闭

### 配置 Git（安装完后做这一次就行）：
1. 按 `Win + R`，输入 `cmd`，回车
2. 在黑窗口里依次输入以下两行（每行输完按回车）：

```
git config --global user.name "BTC520ME"
```
```
git config --global user.email "你的GitHub邮箱"
```

3. 关闭黑窗口

---

## 第三步：把代码上传到 GitHub 🖥️

> 你的仓库已创建：https://github.com/BTC520ME/quxuan-skills

### 3.1 把代码包传到电脑
- 把 `趣选v2.1.0_最终优化版.zip` 通过微信/QQ/网盘发到电脑上
- 解压到桌面（右键 → 解压到当前文件夹）

### 3.2 打开命令行，进入解压后的目录
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入以下命令（直接复制粘贴）：

```
cd %USERPROFILE%\Desktop
```

3. 找到解压后的文件夹名称，输入：

```
dir
```

看看解压后的文件夹叫什么名字，然后进入它（假设叫 `skills`）：

```
cd skills
```

### 3.3 上传到 GitHub
依次输入以下命令，**每行输完按回车，等它执行完再输下一行**：

```
git init
```
↑ 等完成 ↓
```
git add .
```
↑ 等完成 ↓
```
git commit -m "趣选v2.1.0最终优化版 - 6套ClawHub Skill"
```
↑ 等完成 ↓
```
git branch -M main
```
↑ 等完成 ↓
```
git remote add origin https://github.com/BTC520ME/quxuan-skills.git
```
↑ 等完成 ↓
```
git push -u origin main --force
```
↑ 这时会弹出登录窗口，用你的 GitHub 账号登录即可

> ✅ 看到进度条走完就成功了！刷新 GitHub 页面确认文件已上传

## 第四步：发布 Skill 到 ClawHub 🖥️

> 6 套 Skill 需要逐个发布，每个操作一模一样

### 4.1 安装 ClawHub CLI
在黑窗口输入：
```
npm install -g clawhub
```

> 如果提示 npm 不是命令，先安装 Node.js：访问 https://nodejs.org 下载 LTS 版本，一路 Next 安装，然后重开黑窗口

### 4.2 登录 ClawHub
```
clawhub login
```
按提示在浏览器中完成授权

### 4.3 逐个发布（6次操作，每次改个名字）

在代码根目录下，依次执行：

```
clawhub publish quxuan-travel
```
等发布完成 ✅ 再执行下一个 ↓

```
clawhub publish quxuan-daily
```
```
clawhub publish quxuan-workplace
```
```
clawhub publish quxuan-copywriter
```
```
clawhub publish quxuan-professional
```
```
clawhub publish quxuan-revenue-engine
```

> ✅ 每个发布成功后会显示 Skill 的在线地址，记下来！

## 第五步：配置收款方式 💰

### 推荐方案：PayPal + Payoneer 组合

#### 5.1 注册 PayPal（5分钟）
1. 打开 **https://www.paypal.com** → 注册个人账户
2. 用大陆身份证即可注册
3. 绑定一张 Visa/Mastercard 银行卡
4. 注册完成后，记下你的 PayPal 邮箱

#### 5.2 注册 Payoneer（5分钟）
1. 打开 **https://payoneer.com** → 注册个人账户
2. 用身份证注册（无需护照）
3. 完成身份验证（拍身份证正反面+自拍，1-2个工作日通过）
4. 添加 PayPal 为资金来源

#### 5.3 收款流程
```
海外用户付款 → PayPal 收款 → Payoneer 中转（自动换汇）→ 国内银行卡
```

**实际费率**：约 7-9%（PayPal 4.4% + Payoneer 1.2% + 预扣税约10%可退还部分）

> 💡 也可以同时挂到闲鱼/淘宝卖国内用户，0手续费
