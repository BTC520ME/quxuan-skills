# 趣选 ClawHub 收款方式推荐方案

> 适用对象：个人开发者（无营业执照），面向国内外用户

## 核心结论

ClawHub 发布 Skill **免费**（类似 npm），不在平台内收费。
变现通过**外部渠道**卖打包版/服务。

---

## 一、推荐收款方案（按优先级排序）

### 🥇 方案一：PayPal + Payoneer（最适合你）

| 项目 | 说明 |
|------|------|
| **注册门槛** | PayPal：大陆身份证+手机号即可注册个人账户 |
| **收款费率** | PayPal 跨境收款 ~4.4% + 固定费用 |
| **美国扣税** | 默认30%，填 W-8BEN 表降到 **10%**（中美税收协定） |
| **提现方式** | PayPal → Payoneer → 国内银行卡 |
| **换汇费率** | Payoneer 1.2%（直接提现人民币） |
| **综合成本** | 约 **7%-9%**（含PayPal+Payoneer+汇损） |
| **到账时间** | 1-3个工作日 |

**为什么最适合你：**
- 不需要营业执照
- 不需要注册海外公司
- 注册流程最简单
- 全球用户都能付款

### 🥈 方案二：Payoneer 派安盈

| 项目 | 说明 |
|------|------|
| **注册门槛** | 大陆身份证即可 |
| **提现费率** | **1.2%** 到国内银行卡 |
| **优势** | 费率低，支持多平台 |
| **适合** | 大额收款 |

### 🥉 方案三：万里汇 WorldFirst（蚂蚁集团旗下）

| 项目 | 说明 |
|------|------|
| **注册门槛** | 大陆身份证/护照 |
| **提现费率** | **0.4%** 到人民币 |
| **优势** | 费率最低，蚂蚁集团背景安全 |
| **适合** | 长期稳定收款 |

### 🏆 方案四（进阶）：注册美国 LLC + Stripe

| 项目 | 说明 |
|------|------|
| **注册成本** | 首次 $400-700（怀俄明州LLC+EIN+银行） |
| **年维护** | $650-950/年 |
| **Stripe费率** | 2.9% + $0.30/笔 |
| **适合时机** | 年收入超 $5,000 后再考虑 |

## 二、变现渠道推荐

### 渠道1：Gumroad（海外首选）
- 全球用户可购买
- 支持信用卡/PayPal付款
- 抽成 10%
- 无需营业执照
- 注册即用

### 渠道2：闲鱼/淘宝（国内首选）
- 国内用户最方便
- 零门槛，零成本
- 打包成"趣选AI工具包"售卖
- 定价 ¥9.9-99/套

### 渠道3：ClawMart（OpenClaw生态）
- 专门卖 Skill 的第三方商店
- 订阅制 $39-49/月
- 一次性 $99
- 月入 $80,526 的成功案例

### 渠道4：自建微信收款
- 微信群/朋友圈推广
- 微信直接收款
- 零费率，零门槛

## 四、W-8BEN 减税操作要点

1. PayPal 注册后，在税务信息页面填写 W-8BEN
2. 注册主体选 **Individual**（个人）
3. 是否为美国人选 **No**
4. 地址用**拼音**填写中国地址
5. 地址验证失败时勾选 **override** 强制确认
6. TIN 选 **Yes**，Foreign TIN 填你的**中国身份证号**
7. 条约国家选 **China**
8. 预扣税自动从 30% 降到 **10%**
