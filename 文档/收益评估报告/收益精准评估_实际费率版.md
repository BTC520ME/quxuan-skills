## 一、收款渠道实际费率

### 综合费率计算（海外用户）

```
用户付 $10 → PayPal 扣 $0.74 (4.4%+$0.30) → 剩余 $9.26
→ 美国预扣税 10% → 扣 $0.93 → 剩余 $8.33
→ Payoneer 提现 1.2% → 扣 $0.10 → 到手 $8.18

实际到手比例：81.8%（综合费率约 18.2%）
```

### 综合费率计算（Gumroad 渠道）

```
用户付 $10 → Gumroad 扣 10% → 剩余 $9.00
→ PayPal 扣 4.4%+$0.30 → 剩余 $8.26
→ 预扣税 10% → 扣 $0.83 → 剩余 $7.43
→ Payoneer 1.2% → 到手 $7.30

实际到手比例：73.0%（综合费率约 27.0%）
```
