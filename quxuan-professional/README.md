### 📚 法律信息查询
```bash
python scripts/main.py --mode legal \
  --query "公司拖欠工资3个月怎么办"
```

**输出示例（节选）：**
```markdown
## 📚 法律分析：公司拖欠工资

### 🔖 相关法律条款
1. **《劳动法》第50条**：工资应当以货币形式按月支付给劳动者本人。不得克扣或者无故拖欠劳动者的工资。
2. **《劳动合同法》第30条**：用人单位应当按照劳动合同约定和国家规定，向劳动者及时足额支付劳动报酬。
3. **《劳动合同法》第85条**：用人单位未按照劳动合同的约定或者国家规定及时足额支付劳动者劳动报酬的，由劳动行政部门责令限期支付；逾期不支付的，责令用人单位按应付金额50%以上100%以下的标准向劳动者加付赔偿金。

### 🛤️ 维权途径（按推荐顺序）
1. **内部协商** → 与HR/管理层沟通，保留书面记录
2. **劳动监察投诉** → 拨打12333或到当地劳动监察大队投诉
3. **劳动仲裁** → 向当地劳动仲裁委员会申请仲裁（免费）
4. **法院诉讼** → 对仲裁结果不服可向法院起诉

### 📋 证据清单
- [ ] 劳动合同
- [ ] 工资条/银行流水
- [ ] 考勤记录
- [ ] 催款沟通记录（微信/邮件截图）
- [ ] 同事证言（如有）

---
⚠️ 免责声明：以上信息仅供参考，不构成法律建议。具体操作请咨询专业律师。
```

### ⚖️ 劳动仲裁
```bash
python scripts/main.py --mode labor \
  --query "被公司无故辞退，工作3年，月薪15000，没有提前通知"
```

**赔偿计算输出：**
```markdown
## 💰 赔偿金额计算

### 基本情况
- 工作年限：3年
- 月工资：¥15,000
- 解除方式：无故辞退（违法解除）

### 计算结果
| 项目 | 金额 | 计算方式 |
|------|------|---------|
| 经济补偿金 | ¥45,000 | 3年 × 1.5个月 × ¥15,000 |
| 违法解除赔偿金 | ¥90,000 | 经济补偿金 × 2倍 |
| 代通知金 | ¥15,000 | 1个月工资（未提前30天通知） |
| **合计** | **¥105,000** | |

### 仲裁流程
1. 准备材料（1-3天）
2. 提交申请到劳动仲裁委（免费）
3. 仲裁委5日内决定是否受理
4. 受理后45日内出裁决（复杂案件可延期15天）
5. 不服可15日内向法院起诉
```
